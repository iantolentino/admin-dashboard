Quick Start

Learn how to quickly customize the dashboard for your needs.

Changing Colors

Edit the CSS variables in assets/css/style.css:


:root {
    --primary-color: #4e73df;
    --success-color: #1cc88a;
    --info-color: #36b9cc;
    --warning-color: #f6c23e;
    --danger-color: #e74a3b;
}

Adding New Pages

Create new HTML files in the pages/ directory and follow the existing structure.

Dark Mode

The dashboard includes built-in dark mode support. Users can toggle between themes.


// Toggle dark mode
const darkModeToggle = document.getElementById('darkModeToggle');
darkModeToggle.addEventListener('click', function() {
    document.documentElement.setAttribute('data-bs-theme', 
        document.documentElement.getAttribute('data-bs-theme') === 'dark' ? 'light' : 'dark'
    );
});

Dashboard

The main dashboard provides an overview of your application's key metrics.

Components

  • Stats Cards: Display key metrics with progress indicators
  • Charts: Revenue overview and traffic sources
  • Activity Timeline: Recent user activities
  • Performance Metrics: System performance indicators
  • Quick Actions: Direct links to important sections

Customizing Charts

Modify chart data in assets/js/charts.js:


// Revenue Chart
const revenueChart = new Chart(ctx, {
    type: 'line',
    data: {
        labels: ['Jan', 'Feb', 'Mar', ...],
        datasets: [{
            label: 'Revenue',
            data: [12000, 19000, 15000, ...],
            borderColor: '#4e73df',
            backgroundColor: 'rgba(78, 115, 223, 0.1)'
        }]
    }
});

Fetching Data

Example of fetching user data:


async function loadUsers() {
    try {
        const response = await fetch('/api/users');
        const users = await response.json();
        renderUsers(users);
    } catch (error) {
        console.error('Error loading users:', error);
        showToast('Failed to load users', 'error');
    }
}
